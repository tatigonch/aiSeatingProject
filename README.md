# Система рассадки учащихся с ИИ

Веб-приложение для автоматической рассадки учащихся в классе с помощью искусственного интеллекта. ИИ учитывает зрение, рост, конфликты и предпочтения по соседству.

## 🌟 Возможности

- **ИИ-рассадка**: автоматическое размещение учащихся с помощью OpenAI GPT-4o-mini
- **Управление учащимися**: добавление, редактирование, удаление
- **Учёт характеристик**: зрение, рост, конфликты, желаемые соседи
- **Импорт/экспорт**: работа с Excel файлами
- **Визуализация**: современная схема класса с цветовой индикацией
- **Сохранение**: автоматическое сохранение в браузере

## 📋 Характеристики учащихся

- **Зрение**: Хорошее, Среднее, Плохое
- **Рост**: в сантиметрах (140-200 см)
- **Конфликты**: связи с другими учащимися
- **Желаемые соседи**: предпочтительные пары для совместной посадки

## 🤖 ИИ-алгоритм рассадки

Приложение использует **OpenAI GPT-4o-mini** для интеллектуальной рассадки. ИИ получает данные обо всех учениках и применяет следующие правила (в порядке приоритета):

1. Ученики с плохим зрением размещаются в первых рядах (ближе к доске)
2. Высокие ученики — в задних рядах или по краям, чтобы не загораживать другим
3. Конфликтующие ученики не сидят за одной партой (строгое ограничение)
4. Желаемые соседи по возможности сидят вместе (мягкое ограничение)
5. Ученики с плохим зрением и высоким ростом — в первый ряд по краям

## 🛠 Технологии

- **Frontend**: React 18, Material-UI
- **ИИ**: OpenAI API (GPT-4o-mini)
- **Файлы**: SheetJS (Excel), jsPDF
- **Хранение**: localStorage
- **Деплой**: GitHub Pages

## 🏃‍♂️ Запуск локально

```bash
npm install
npm start
```

### Настройка API-ключа

Для работы ИИ-рассадки необходим ключ OpenAI API. Создайте файл `.env` в корне проекта:

```env
REACT_APP_OPENAI_API_KEY=ваш-ключ-api
```

Получить ключ можно на [platform.openai.com](https://platform.openai.com).
